Observation identification service API

De Observation identification service API is onderdeel van de Naturalis Biodiversity Analysis services.

Via deze API kun je

  • beperkt gebruik maken van het soortherkeningsmodel van waarneming.nl
  • inzicht krijgen in de taxa die het model kan herkennen
  • documentatie opvragen over de API
Wil je inzage in de complete API documentatie download dan de Observation Identification Service API specifications (PDF).
